# Tech Debt

Техдолг ведем отдельными тикетами в `docs/debt/`.

Формат имени:

- `TD-001-short-name.md`
- `TD-002-short-name.md`

Рекомендуемая структура тикета:

- Контекст
- Правильная модель
- Что сейчас не так
- Что уже пробовали
- Следующий заход
- Зависимости/риски

Правила:

- Один тикет = одна проблема или один связанный узел поведения
- Если запись в `docs/TODO.md` требует длинного контекста, лучше выносить ее сюда
- В `docs/TODO.md` можно оставлять короткую ссылку на тикет
